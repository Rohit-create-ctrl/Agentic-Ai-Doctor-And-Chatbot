# 🩺 Agentic AI Doctor & Chatbot

A full-stack AI-powered medical assistant and health monitoring dashboard built with Next.js, Express.js, and MongoDB.

## 🚀 Features
- **AI Chatbot**: Symptom analysis and medicine recommendations.
- **Health Dashboard**: Track medical records and history.
- **Medicine Database**: Extensive information on common medications.
- **Emergency Support**: Quick access to emergency services and advice.

## 📁 Project Structure
- `/ai-doctor`: Next.js Frontend
- `/backend`: Express.js & MongoDB Backend

## 🛠️ Setup Instructions

### Backend
1. `cd backend`
2. `npm install`
3. Create a `.env` file based on `.env.example`
4. `npm run dev`

### Frontend
1. `cd ai-doctor`
2. `npm install`
3. Create a `.env.local` file based on `.env.example`
4. `npm run dev`

## 🌐 Deployment
- **Frontend**: Deploy the `/ai-doctor` directory to Vercel.
- **Backend**: Deploy the `/backend` directory to Render/Railway.
- **Database**: Use MongoDB Atlas.
